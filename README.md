# calculator_project
This is a CLI calculator

To run it:
download the last version of C file, compile it with:
cc _name_of_the_file -lm
and run:
./a.out

Example:

root@Mobile1516T:/home# ./a.out "(((3.14*3^2) * 5)/3) + (((3.14*3^2) * 5)/3)"
Result: 94.199997
